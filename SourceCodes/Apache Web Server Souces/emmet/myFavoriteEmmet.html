Syntax : li>a>{텍스트 입력, href 값 미지정시 현재 사이트의 Default로 링크됨}

  • 텍스트 입력

    • Syntax : ul>li.item$*5 일련번호 숫자 삽입은 $ Sign만 추가하면 됨

    Syntax : li>a[href="http://www.emmet.org/" target="_blank"]>{텍스트 입력}

  • 숫자 자동 증가, 자기만의 반복 코드 단축키로 만드는 방법은 Emmet Syntax를 보라.

    • Syntax : a{click }+b{here}

    click here

    Syntax : a>{click }+b{here}+{ to continue}

    click here to continue
